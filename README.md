# 厦门亲像一首歌 - 歌词播放器

一个简洁的闽南语歌曲歌词同步播放器，支持音频播放和歌词自动高亮。

## 功能特点

- 🎵 音频播放控制
- 📝 歌词自动高亮和滚动
- ⏱️ 歌词时间校准功能
- 🎯 点击歌词跳转播放位置
- 📱 响应式设计，支持移动端

## 使用说明

1. 打开 `lyrics-player.html` 文件
2. 开始播放音频
3. 如需调整歌词同步时间，使用"提前"或"延后"按钮微调

## 文件说明

- `lyrics-player.html` - 主播放器页面
- `lyrics.md` - 歌词文本文件
- `20251026_214808.m4a` - 音频文件
